# StudySync Project Roadmap

## Project Description
StudySync is a Python-based application that helps students track assignments, due dates, and study sessions. Users can input tasks, view upcoming work, and stay organized.

## Target Users
College students who want a simple way to manage homework and deadlines.

## Features (Planned)
- Add assignments
- View assignments
- Mark assignments as complete
- Store assignments in a file
- Simple menu system
